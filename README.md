# Bitcoin Trader Sentiment Analysis

## Objective
Analyze relationship between trader performance and market sentiment (Fear vs Greed).

## Key Insights
- Traders performed better during Greed periods
- Higher leverage observed during Greed
- Increased trading activity during Fear

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- Hyperliquid Trader Data
- Bitcoin Fear & Greed Index
